## 確認済みで問題なし

- 内部リンク、相対画像参照、見出しアンカーは機械チェックで破損なしでした。
- [00-PREPARE.md](00-PREPARE.md) の3Dプリント部品リンクは、現在の `models/` 配下の `.stl` ファイルと対応しています。
- キー数の説明は、準備ガイド、組み立てガイド、ファームウェア設計の間で「70キー、左31 + 右39」と整合しています。
- [02-FIRMWARE.md](02-FIRMWARE.md) と [05-CUSTOMIZE.md](05-CUSTOMIZE.md) のUF2ファイル名は、`build.yaml` のビルド対象と整合しています。
- リリースページから直接UF2をダウンロードする説明と、GitHub Actionsの `firmware` artifactを展開する説明は、文脈が異なるため食い違いではありません。
- [04-USAGE.md](04-USAGE.md) の `Ctrl` + `Fn` + `1` から `5` と `Ctrl` + `Fn` + `Backspace` は、`cleave_hhjp.keymap` の `bt1` から `bt5`、`btclr` の定義と整合しています。
- ZMK Studioを左側キーボードで使う説明は、左側のみ `CONFIG_ZMK_STUDIO=y` になっている設定と整合しています。
- LiPoバッテリーの注意、`settings_reset`、M2ネジ長の再掲は、安全確認や別章からの参照時に必要な内容で、削るより残す方が親切です。
