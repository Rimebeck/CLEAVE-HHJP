# CLEAVE HHJP ビルドガイド

CLEAVE HHJP の組み立て、ファームウェア書き込み、ケース作成、完成後の使い方、カスタマイズ方法をまとめたガイドです。

> [!WARNING]
> CLEAVE HHJPを組み立てる際は、関連するガイドをよく読んでから作業を開始してください。
> ガイドを遵守しない場合、故障や事故の原因となる可能性があります。

## 概算費用

総額：¥20,000~

工具代と送料を除いた目安です。価格は購入先、在庫、為替、MX/Choc両対応か、選ぶケース素材によって大きく変わります。


| 項目 | 目安 | 内容 |
|---|---|---|
| キット | 頒布価格を確認してください | PCB、ダイオード、電源スイッチ、バッテリーコネクタ、ケース用ネジ類など |
| マイコン | 4,000円程度 | XIAO nRF52840 Plus 2個 |
| 3Dプリント部品 | 自家プリント: 約1,000〜3,000円<br>外注: 約8,000〜10,000円 | 3Dプリント部品が必要な部品一式 |
| ユーザーが独自に用意するもの | 〜5,000円 | LiPoバッテリー、ホットスワップソケット、フォームテープ、滑り止めなど |

※キースイッチおよびキーキャップは上記に含まれません。

## ビルドの進め方

このガイドでは、必要なものをまとめて準備したあと、基板を組み立て、ファームウェアを書き込んで基板単体で動作確認し、その後ケースに組み込む流れを標準ルートとします。

> [!NOTE]
> ガイド内の画像には、開発途中の基板やケースが写っているものが含まれます。
> そのため実際の見た目と完全には一致しない場合がありますが、説明している箇所には影響ありません。

| 順番 | ガイド | 作業の位置付け |
|---|---|---|
| 0 | [準備ガイド](00-PREPARE.md) | 購入・発注前に決めること、必要な工具、電子部品、ケース部材、3Dプリント部品を確認します。 |
| 1 | [基板組み立てガイド](01-ASSEMBLY.md) | 準備した部品を基板へ実装し、ファームウェア書き込み後の基板単体確認に備えてキースイッチとバッテリーを接続します。 |
| 2 | [ファームウェアのダウンロード・書き込み・接続確認](02-FIRMWARE.md) | 左右それぞれのXIAO nRF52840 PlusへUF2ファイルを書き込み、基板単体で動作確認します。 |
| 3 | [ケースビルドガイド](03-CASE.md) | 動作確認済みの基板をケースへ組み込み、キースイッチとバッテリーを最終固定します。 |
| 4 | [完成後の使い方](04-USAGE.md) | ケース組み立て後の最終確認、電源、Bluetooth接続、充電方法を確認します。 |
| 5 | [カスタマイズガイド](05-CUSTOMIZE.md) | 必要に応じてキーマップやファームウェア設定を変更します。 |

ケースなしの状態は、ファームウェア書き込みや不具合切り分けのための動作確認用です。常用する場合は、ケース組み立て後に使用することを推奨します。
