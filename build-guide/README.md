# Cleave HHJP ビルドガイド

Cleave HHJP の組み立て、ファームウェア書き込み、ケース作成、完成後の使い方、カスタマイズ方法をまとめたガイドです。

> [!WARNING]
> Cleave HHJPを組み立てる際は、関連するガイドをよく読んでから作業を開始してください。
> ガイドを遵守しない場合、故障や事故の原因となる可能性があります。

## 目次

> [!NOTE]
> ガイド内の画像には、開発途中の基板やケースが写っているものが含まれます。
> そのため実際の見た目と完全には一致しない場合がありますが、説明している箇所には影響ありません。

1. [基板組み立てガイド](01-ASSEMBLY.md)  
   必要な道具と部品、基板への部品実装、基板単体での動作確認用手順
2. [ファームウェアのダウンロード・書き込み・接続確認](02-FIRMWARE.md)  
   UF2ファイルの入手、XIAO nRF52840 Plusへの書き込み、Bluetooth接続確認
3. [ケースビルドガイド](03-CASE.md)  
   配布されている3Dモデルを使用したケース作成と組み立て
4. [完成後の使い方](04-USAGE.md)  
   電源、Bluetooth接続、ペアリングリセット、充電
5. [カスタマイズガイド](05-CUSTOMIZE.md)  
   ZMK Studioでのキーマップ変更、リポジトリをフォークして行うファームウェア変更
