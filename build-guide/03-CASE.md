## ケースの選択肢

Cleave HHJP のケースを作る方法は大きく分けて2つあります。

1. 配布されているデフォルトケースの3Dモデルを使用する  
   自分で3Dプリントする、またはJLC3DPなどの3Dプリントサービスに発注して作成します。
2. PCBのSTEPファイルを使用してオリジナルケースを設計する  
   ユーザー自身でケースを設計し、好みの構造や素材に合わせて作成します。

このガイドでは、1つ目の「配布されているデフォルトケースの3Dモデルを使用する」方法を説明します。

## デフォルトケースの構成

デフォルトケースは、ボトムケース、PCB、トッププレート、トップカバー、キースイッチなどを組み合わせて作成します。

## 用意するもの

配布されているケースの3Dモデルを使用してケースを作成する場合は、以下の部品を用意します。

### 3Dプリントする部品

配布されているデフォルトケースの3Dモデルから、以下のケース部品を作成します。トッププレートは使用するキースイッチに合わせて選択し、トップカバーはインジケーター穴の有無を選択します。

**共通で使用する部品**
|役割|部品名|数量|選択条件|備考|
|---|---|---|---|---|
|ボトムケース|左用/右用ボトムケース|左右各1個|共通|MX/Choc共通|
|トップカバー|左用/右用トップカバー|左右各1個|インジケーター穴あり/穴なしから選択|MX/Choc共通|
|電源スイッチノブ|左用/右用電源スイッチノブ|左右各1個|共通|ケース外側から電源スイッチを操作するために使用|

**MX互換スイッチを使用する場合**
|役割|部品名|数量|備考|
|---|---|---|---|
|トッププレート|MX用トッププレート|左右各1枚|MX互換スイッチ用|

**ChocV1/V2互換スイッチを使用する場合**
|役割|部品名|数量|備考|
|---|---|---|---|
|トッププレート|Choc用トッププレート|左右各1枚|ChocV1/V2互換スイッチ用|

### 3Dプリント部品以外で用意する部材

3Dプリントしたケース部品に加えて、以下の部材も必要になります。

|役割|部品名|数量|キット付属|備考|
|---|---|---|---|---|
|インサートナット||14個|○||
|ネジ|M2 8mm|8本|○|Chocトッププレート用|
|ネジ|M2 10mm|8本|○|MXトッププレート用|
|ネジ|M2 4mm|6本|○|トップカバー用|
|インジケーターランプ|直径2mm透明プラ棒（3mm程度）|4本|○|インジケーター穴ありトップカバーを使う場合に使用。穴なしまたは透明トップカバーを使う場合は不要|
|フォームテープ|フォームテープ2mm厚/3mm幅|2m程度||幅広のテープをカットして使う事も可能です。|
|滑り止めシート|ポロンシートなど|||任意|

## ケース組み立て手順

> [!NOTE]
> **左右両方で作業してください**
>
> このガイドでは写真や説明が片側のみの場合がありますが、特に記載がない限り、同じ手順を左用・右用の両方で行ってください。

### 1. 3D プリント部品の準備
配布されている3Dモデルを使い、ケース組み立てに必要な部品を用意します。モデル名やフォルダー構成は配布時点の内容に従い、印刷前に以下の構成を決めてください。

**印刷前に決めること**

- **使用するキースイッチの種類**：MX互換スイッチ、またはChocV1/V2互換スイッチを選びます。選択したスイッチに合わせてトッププレートを選び、固定にはMX用でM2 10mm、Choc用でM2 8mmのネジを使用します。
- **トップカバーのタイプ**：インジケーター穴あり、または穴なしを選びます。穴ありを選んだ場合は「9. インジケーターの取り付け」で透明プラ棒を取り付けます。穴なしを選んだ場合、その工程は実施不要です。

**印刷する部品**

|部品|必要数|選択条件|素材・注意点|
|---|---|---|---|
|ボトムケース|左1個、右1個|共通|インサートナットを熱圧入するため、FDMプリント用の樹脂、またはMJF/SLSのナイロン系など、熱可塑性のある素材を選びます。|
|トップカバー|左1個、右1個|インジケーター穴あり/穴なしから選択|インサートナットを圧入しないため、レジン系素材も選択肢になります。|
|電源スイッチノブ|左1個、右1個|共通|インサートナットを圧入しないため、レジン系素材も選択肢になります。ノブの摺動部に造形残りがないことを確認します。|
|トッププレート|左1枚、右1枚|MX用またはChoc用から選択|インサートナットを圧入しないため、レジン系素材も選択肢になります。使用するスイッチに合うプレートだけを用意します。|

[ケースの3Dモデル](https://github.com/07JP27/Cleave-HHJP/tree/main/models) から、決めた構成に対応するモデルを選んで3Dプリントします。

3Dプリントサービスに発注する場合は、部品ごとの用途に合わせて素材を選択してください。

- ボトムケースはインサートナットを熱圧入するため、FDMプリント用の樹脂、またはMJF/SLSのナイロン系など、熱可塑性のある素材を選びます。
- トップカバー、電源スイッチノブ、トッププレートはインサートナットを圧入しないため、レジン系素材も選択肢になります。
- ねじ穴、インサートナット穴、スイッチ穴、電源スイッチノブの摺動部、トップカバーのインジケーター穴にサポート材や粉が残っている場合は、組み立て前に取り除きます。

**完了チェック**

- [ ] 使用するキースイッチの種類（MX / Choc）を決めた
- [ ] トップカバーのタイプ（インジケーター穴あり / 穴なし）を決めた
- [ ] 左右のボトムケースを用意した
- [ ] 左右のトップカバーを、選択したタイプで用意した
- [ ] 左右の電源スイッチノブを用意した
- [ ] 使用するキースイッチに対応した左右のトッププレートを用意した
- [ ] ねじ穴、インサートナット穴、スイッチ穴、電源スイッチノブの摺動部に造形残りがない
- [ ] インジケーター穴ありトップカバーを選んだ場合、左右のトップカバーにインジケーター穴が開いている

### 2. インサートナットの取り付け
左右7箇所にインサートナットをハンダごてで熱圧入します。
ナットの上面とケースの上面が揃うところまで押し込みます。
<img src="./images/case/02/2.jpg" width="600px">

**完了チェック**

- [ ] 左右それぞれ7箇所にインサートナットを取り付けた
- [ ] インサートナットの上面がケースの上面から大きく飛び出していない
- [ ] インサートナットが傾いていない
- [ ] ネジを軽く入れて、無理なく回ることを確認した


### 3. フォームテープの貼り付け
インサートナットを避けてベースの梁の部分にフォームテープを貼ります。
<img src="./images/case/03/1.jpg" width="600px">

**完了チェック**

- [ ] フォームテープがケース内側の梁に沿って貼られている
- [ ] インサートナットやネジ穴をフォームテープで塞いでいない

### 4. キースイッチ取り付け
取り付け前に、使用するキースイッチに合ったトッププレートを用意していることを確認してください。MX互換スイッチとChocV1/V2互換スイッチでは、使用するトッププレートが異なります。
MX互換スイッチで2Uキーを使用する場合は、この工程でPCBマウント2Uスタビライザーも取り付けます。スタビライザーを取り付ける位置は、使用するキーキャップ構成に合わせて確認してください。
<img src="./images/case/04/1.jpg" width="600px">

**完了チェック**

- [ ] 使用するキースイッチに合ったトッププレートを用意した
- [ ] スイッチのピンが曲がらず、PCBのソケットに入っている
- [ ] MX構成で必要な場合は2Uスタビライザーを取り付けた
- [ ] スイッチがトッププレートにまっすぐ収まっている


### 5. トッププレートの取り付け
PCBにスイッチを差し込みながらサンドイッチします。
<img src="./images/case/05/1.jpg" width="600px">

**完了チェック**

- [ ] スイッチがトッププレートとPCBの両方に正しく入っている
- [ ] トッププレートが大きく反っていない
- [ ] ネジ穴の位置がPCB、トッププレート、ボトムケースで合っている

### 6. 電源スイッチノブの取り付け
ケースの内側から電源スイッチノブを差し込みます。

| 電源スイッチノブをケースの裏側から通して→ | ケースの外側にノブを出す |
|---|---|
| <img src="./images/case/06/1.jpg" width="100%"> | <img src="./images/case/06/2.jpg" width="100%"> |

**完了チェック**

- [ ] 電源スイッチノブがケース外側へ出ている
- [ ] ノブが引っかからず左右に動く
- [ ] PCBを取り付けたときに、基板上の電源スイッチとノブが噛み合う向きになっている


### 7. PCBの取り付け
ベースケースにPCBとトッププレートとキースイッチが一体になった部品を装着します。
電源スイッチの窪みにPCBの電源スイッチのノブが刺さるようにスライドさせながらはめ込みます。

<img src="./images/case/07/1.jpg" width="600px">

トッププレートのフチがベースケースにぴったり合うように調整し、左右それぞれ4箇所を 長さ8(Choc)/10(MX)mmのM2ネジで止めます。

<img src="./images/case/07/position.png">

**完了チェック**

- [ ] PCB、トッププレート、キースイッチがボトムケースに収まっている
- [ ] 電源スイッチノブで基板上の電源スイッチを操作できる
- [ ] トッププレートのフチがボトムケースに沿っている
- [ ] 左右それぞれ4箇所を、使用するトッププレートに合った長さのM2ネジで固定した
- [ ] ネジを締めすぎてケースやトッププレートが変形していない


### 8. バッテリーの固定
バッテリーをコネクタに差し込んでバッテリー用のスペースに格納します。
バッテリーが振動などで破損しないように、バッテリー裏面を両面テープなどでケースに固定することを強く推奨します。

<img src="./images/case/08/1.jpg" width="600px">

**完了チェック**

- [ ] バッテリーコネクタが奥まで差し込まれている
- [ ] バッテリーがケース内のスペースに収まっている
- [ ] バッテリーやケーブルがネジ穴、トップカバー、PCBに挟まれない
- [ ] バッテリーがケース内で動かないように固定されている

### 9. インジケーターの取り付け
> [!NOTE]
> 穴なしトップカバーを使う場合はこのステップは実施不要です。
直径2mm透明プラ棒（3mm程度）をトップカバーに差し込みます。
接着はまだしません。

<img src="./images/case/09/1.jpg" width="600px">

**完了チェック**

- [ ] 透明プラ棒を左右それぞれ必要な位置に差し込んだ

### 10. トップカバーの取り付け
トップカバーをベースケースに3箇所 4mmのM2ネジ止めします。
<img src="./images/case/10/position.png">

インジケーターを付けた場合はここでPCBとインジケーターの隙間を調整し、上面に出ている不要な分はニッパーなどで切って調整したあとに接着剤などを使ってカバーの側から接着してください。
<img src="./images/case/10/2.jpg" width="600px">

**完了チェック**

- [ ] トップカバーを左右それぞれ3箇所のM2 4mmネジで固定した
- [ ] トップカバーが浮かず、ボトムケースに沿っている
- [ ] インジケーターを付けた場合は、透明プラ棒の長さを調整して固定した
- [ ] 電源スイッチノブがトップカバーと干渉せずに動く


### 11. キーキャップの取り付け
用意したキーキャップをスイッチに装着します。

**完了チェック**

- [ ] すべてのキースイッチにキーキャップを取り付けた
- [ ] キーキャップが隣のキーやケースに干渉していない
- [ ] スタビライザーを使うキーが傾かずに押せる

### 12. 滑り止めの取り付け
3Dプリントしたケースや使うデスクの材質によってはキーボードが滑る事があるので、お好みに応じてケースの裏側に両面テープなどでポロンシートやすべり止めシートを貼り付けます。

**完了チェック**

- [ ] 左右のケース裏面に滑り止めを貼り付けた
- [ ] 滑り止めが机に接地している
- [ ] キーボードを置いたときにがたつきがない
